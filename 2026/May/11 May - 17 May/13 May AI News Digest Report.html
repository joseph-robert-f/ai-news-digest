Google DeepMindAI co-mathematician sets new FrontierMath Tier 4 high; cracks 60-year Kourovka problem

12 May 2026 · Primary source: Google DeepMind / arXiv 2605.06651

Google DeepMind’s AI co-mathematician, a stateful multi-agent workbench built on Gemini 3.1 Pro with Gemini Deep Think as a sub-agent, scored 48% on FrontierMath Tier 4 — Epoch AI’s hardest research-grade benchmark, designed to remain unsolved by AI “for decades.” The base Gemini 3.1 Pro scores 19% on the same set. More striking than the leaderboard: Oxford’s Marc Lackenby used the system to resolve Problem 21.10 from the Kourovka Notebook, an open group-theory question that had stood for roughly 60 years. The system’s reviewer agent flagged a flaw in its own first proof, which Lackenby then filled. The paper lists the system ahead of GPT-5.5 Pro (39.6%), GPT-5.4 Pro (37.5%), and Claude Opus 4.7 (22.9%) on the same benchmark.

The AI Cybersecurity Race

OpenAIDaybreak launches with GPT-5.5-Cyber and Codex Security; Anthropic Mythos rivalry sharpens

11–12 May 2026 · Primary source: OpenAI / Sam Altman

OpenAI publicly launched Daybreak, a cyber-defense initiative organized around three pieces: the broad GPT-5.5 base model for general security work, GPT-5.5-Cyber (a specialized model under Trusted Access for red-teaming and pen-testing), and a Codex Security agent framework for secure code review, vulnerability triage, malware analysis, detection engineering, and patch validation. Daybreak ships with eight named partners — Cisco, Oracle, CrowdStrike, Palo Alto Networks, Cloudflare, Fortinet, Akamai, and Zscaler. OpenAI’s framing is “resilient by design”: bake security into software before exploitation. Access remains gated to vetted defenders for now. The launch is explicitly OpenAI’s answer to Anthropic’s Project Glasswing / Claude Mythos, which Mozilla credited with finding and patching 271 Firefox vulnerabilities last month.

Google GTIGFirst AI-generated zero-day in the wild; 2FA bypass disrupted before mass-exploitation

11 May 2026 · Primary source: Google Cloud Threat Intelligence Group

Google’s Threat Intelligence Group (GTIG) said it has “high confidence” it has now observed the first AI-generated zero-day exploit in active criminal use. The flaw — a 2FA-bypass in a Python script used by an open-source web-based sysadmin tool — was discovered and weaponized using an AI model (Google says it does not believe its own Gemini was used). A criminal threat actor was preparing it for a mass exploitation event; GTIG’s proactive counter-discovery likely prevented use. The disclosure ties into GTIG’s expanded AI-misuse report: malware families PROMPTFLUX (VBScript dropper that queries Gemini for “just-in-time” obfuscation) and PROMPTSTEAL (deployed by Russia’s APT28 against Ukraine, using Qwen2.5-Coder-32B via Hugging Face) are now in active operations. GTIG chief analyst John Hultquist: “The era of AI-driven vulnerability discovery and exploitation is already here.”
