OpenAIInternal reasoning model disproves Erdős' 1946 unit-distance conjecture — first autonomous breakthrough on a central open problem

20–21 May 2026 · Primary source: OpenAI Research / arXiv 2605.20695

OpenAI announced this week that an internal, general-purpose reasoning model — not a math-specialised system — autonomously disproved Paul Erdős' 1946 conjecture on the maximum number of unit-distance pairs among n points in the plane. For nearly 80 years the conjecture said the optimum was essentially a perturbed square grid; the model discovered an infinite family of constructions that polynomially beats the grid, building the proof out of Golod–Shafarevich infinite class field towers — a connection from algebraic number theory that no human paper had drawn to this problem. The proof has been independently checked by external mathematicians, and Fields medallist Tim Gowers described the result as "a milestone in AI mathematics." Critically, the model was scaffolded only with general proof-search prompts — no Lean tactics, no targeted curriculum — which makes the result a stronger data point on emergent research capability than DeepMind's earlier AlphaProof releases.

Supply ChainNx Console VS Code extension breach widens: ~3,800 GitHub repos exfiltrated; payload targets Claude Code credentials

19–22 May 2026 · Primary sources: Nx security advisory, GitHub disclosure

Fallout continued through the weekend from the nrwl.angular-console v18.95.0 compromise. The malicious build of the popular Nx Console VS Code extension (>2.2 M installs) was published 18 May and pulled within ~11 minutes — long enough for a 498 KB obfuscated payload, hidden inside a dangling orphan commit in the official nrwl/nx GitHub repository, to fire on workspace-open and harvest tokens from GitHub, npm, AWS, HashiCorp Vault, Kubernetes, and 1Password. The payload then attempted to use stolen npm OIDC tokens with full Sigstore / Fulcio / SLSA provenance integration to ship signed downstream packages. On 19 May, GitHub publicly disclosed that ~3,800 internal repositories were exfiltrated. Security teams flagged a notable first: the payload specifically reads ~/.claude/settings.json and related AI-coding-assistant configuration files — the earliest publicly observed example of a supply-chain attack designed to harvest AI agent credentials. TeamPCP has claimed attribution and is attempting to sell the stolen data.
