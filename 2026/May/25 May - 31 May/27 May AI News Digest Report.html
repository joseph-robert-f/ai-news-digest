AnthropicProject Glasswing update: Claude Mythos has found 10,000+ critical vulnerabilities in one month — with about 50 partner orgs on board

26 May 2026 · Primary source: Anthropic / CSO Online / Help Net Security

Anthropic disclosed a major update to Project Glasswing, its defensive-cybersecurity consortium built around the restricted Claude Mythos Preview model. In its first month behind a closed-access window, Mythos has surfaced more than 10,000 high- or critical-severity vulnerabilities across widely deployed open-source software. The disclosure also formalises a previously partial partner list: Amazon Web Services, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Linux Foundation, Microsoft, NVIDIA, and Palo Alto Networks, with around 50 organisations total holding monitored access to the model. In the disclosed numbers, Mythos analysed code across more than 1,000 open-source projects, flagged 6,202 candidate findings in the first month, of which 1,726 were validated by humans as real, exploitable flaws; only 97 have been patched. The takeaway from Anthropic and the partners is that defensive-AI throughput is now outrunning the global patching pipeline by an order of magnitude. Anthropic also reiterated that Mythos is so capable at autonomous vulnerability discovery and exploitation that it will not be released to the public — a position Palo Alto Networks corroborated last week when the same model produced an unusually large 26-CVE advisory bundle.

GoogleGemini Interactions API: new steps schema is now the default — legacy schema removed 8 June

26 May 2026 · Primary source: Google AI for Developers (Gemini API docs)

A scheduled breaking change to Gemini's Interactions API (v1beta) flipped today: the new steps array replaces the legacy outputs array as the default response shape, and a new polymorphic response_format consolidates output-format controls, retiring response_mime_type. The change is in service of mid-flight steering and asynchronous tool calls — the steps array now carries structured, typed step records (user_input, model_output, tool invocations) that an agent harness can stream, reorder, or cancel. Critically, the legacy schema is being removed entirely on 8 June 2026, with a temporary opt-out via the Api-Revision: 2026-05-07 header. SDKs at version ≥ 2.0.0 are already on the new schema; any application still hard-coded to the old outputs array, or relying on response_mime_type for JSON/MIME enforcement, now has a hard 12-day window to migrate. The combination of Gemini 3.5 Flash going GA earlier this month (covered in the 25 May digest) and the schema cutover today functionally completes the Gemini 3.5 platform migration for the developer surface.
